# Django E-Commerce Website

A full-stack e-commerce application built using Django.

## Features
- Product catalog with categories
- Session-based cart
- User authentication
- Checkout & order system
- Dummy payment flow
- Order history

## Tech Stack
- Django
- SQLite (dev)
- Bootstrap
- Razorpay (planned)

## Setup
```bash
pip install -r requirements.txt
python manage.py runserver
